#  Bright Motors: Car Sales Performance Analysis

##  Project Overview
This repository presents a comprehensive data analysis project for **Bright Motors**, a car dealership group aiming to optimize its sales strategy.  

The project demonstrates an end-to-end data workflow—transforming raw transactional data into actionable business insights—using SQL for data engineering and Lovable for interactive visualization.

##  Project Overview
This repository presents a comprehensive data analysis project for **Bright Motors**, a car dealership group aiming to optimize its sales strategy.  

The project demonstrates an end-to-end data workflow—transforming raw transactional data into actionable business insights—using SQL for data engineering and Lovable for interactive visualization.

##  Key Objectives
- **Identify Sales Peaks**  
  Analyze time-series data to uncover high-performing sales periods.

- **Dealer Benchmarking**  
  Compare regional performance to identify top-performing dealerships.

- **Customer Segmentation**  
  Explore relationships between customer income and vehicle purchase behavior.

- **Inventory Optimization**  
  Identify popular vehicle attributes (body style, fuel type) to guide stock decisions.

---

##  Tech Stack & Tools
- **SQL**  
  Used for ETL (Extract, Transform, Load), data cleaning, and null handling.

- **Snowflake**  
  Cloud data warehouse for storage and scalable querying.

- **Lovable**  
  AI-powered platform for building interactive analytics dashboards.

- **Miro**  
  Visualization of data architecture and pipeline design.
